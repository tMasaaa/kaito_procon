# AtCoder ABC 112 D Partition

  • 約数全列挙ができずに時間を食ってしまった

問題

a1+a2+...+aN=Ma_1 + a_2 + ... + a_N = Ma​1​​+a​2​​+...+a​N​​=Mで、GCD(a1,a2,...,aN)GCD(a_1,a_2,...,a_N)GCD(a​1​​,a​2​​,...,a​N​​)の最大値を求めよ

  • 方針
  • 問題文のGCDGCDGCDをgggとおく。左辺はgggで割れるので、g∣Mg \mid Mg∣Mである。つまり、gggはMMMの約数。
  • 約数全列挙をする。
  • NNN個の整数にgggを割り振ることを考えると、MMMの約数のうちNNNを超える整数SSSのうち最小のものをとり、SSS個のgggをNNN個のaia_ia​i​​に適当に割り振れば、g=M/Sg=M/Sg=M/Sとなり、gggが最大化できる。
  • 以上をコードにする。計算量はO(N)O(\sqrt{N})O(√​N​​​)
#include <bits/stdc++.h>
using namespace std;
#define LINF 1000000000000000001 // 1e18+1

int64_t N,M,i;

vector< int64_t > ediv(int64_t n){
  vector< int64_t > ret;
  for(i = 1;i*i <= n;i++){
    if(n%i == 0){
      ret.push_back(i);
      if(i != 1 && i*i != n){
        ret.push_back(n/i);
      }
    }
  }
  ret.push_back(n);
  return ret;
}
int64_t mn = LINF;
int main(){
  std::ios::sync_with_stdio(false);
  std::cin.tie(0);

  // ifstream in("input.txt");
  // cin.rdbuf(in.rdbuf());

  cin >> N >> M;
  for(auto i: ediv(M)){
    if(i >= N){
      mn = min(mn, i);
    }
  }
  cout << M/mn << endl;
  return 0;
}
